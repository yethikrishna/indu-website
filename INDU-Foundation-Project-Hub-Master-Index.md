## What Is INDU?

INDU is a compiled, multi-paradigm programming language and full-stack framework operating on the exact same AST. Write with Python's ease, scale with Erlang's fault-tolerance, execute with C's speed.

**Core primitives:** `agent`, `tensor`, `stream`, `sync let` (CRDT), ARC memory, WebGPU rendering, NEXUS NPU/GPU engine.

**Replaces:** Python + Node.js + React + Next.js + LangChain — in a single unified architecture.

***

## Website Ecosystem Overview

The INDU Foundation website is a **100+ page cinematic ecosystem** built in SvelteKit 2.x + Svelte 5. It rivals activetheory.net in animation quality and rust-lang.org in documentation depth.

### Technology Stack

* **Frontend:** SvelteKit 2.x + Svelte 5 (Runes)

* **3D/Animation:** Three.js + GSAP ScrollTrigger

* **Styling:** CSS Custom Properties (no Tailwind)

* **Database:** Supabase (PostgreSQL + Auth + Realtime)

* **Email:** Resend

* **Deployment:** Cloudflare Pages

* **Search:** Algolia DocSearch

* **Monitoring:** Sentry + Cloudflare Analytics

## Key Design Decisions

1. **SvelteKit over Next.js** — Smaller bundle, no hydration errors, native SSR
2. **Cloudflare Pages over Vercel** — Edge network, no cold starts, better DX for open source
3. **Supabase over Firebase** — Open source, PostgreSQL, row-level security
4. **CSS Custom Properties over Tailwind** — Smaller bundle, full creative control for cinematic effects
5. **Three.js IcosahedronGeometry** — The 3D central object: subdivided icosahedron with aurora vertex shader
6. **GSAP ScrollTrigger** — 5000vh scroll journey mapping 7 narrative sections
7. **Resend over SendGrid** — Developer-first, modern DX, React Email templates
